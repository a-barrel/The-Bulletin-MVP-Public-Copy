# Settings Playbook

## Tabs/Sections
- **Appearance** – theme picker, text scale slider, motion/high-contrast toggles, map density presets, celebration sound toggle (writes to `preferences.display.*`).
- **Notifications** – granular switches for proximity, friend requests, badge unlocks, moderation alerts, DM mentions, marketing, and chat transitions plus the digest frequency selector.
- **Privacy** – location sharing toggle, stats visibility, profanity filter, DM-permissions radio, and the blocked-users overlay.
- **Data & Security** – data export reminders + POST `/api/users/me/data-export`, API token labeler/generator, token listing, revoke controls.
- **Feedback** – anonymous feedback dialog and admin shortcuts (blocked users, profile preview, admin dashboard button when applicable).

Keep this playbook updated as we add new toggles.
